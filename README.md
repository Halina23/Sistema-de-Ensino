### Sistema de Ensino — Banco de Dados


Projeto desenvolvido com foco na estruturação e gerenciamento de dados de um sistema educacional.

Modelagem completa do banco (alunos, professores, cursos, turmas, matrículas e notas)
Criação das tabelas em MySQL com chaves primárias e estrangeiras
Normalização e aplicação de regras de integridade
Desenvolvimento de consultas SQL para relatórios acadêmicos
Administração básica: usuários, permissões, backup e restore
Manipulação e testes de dados com Python
Desenvolvimento no VS Code.
